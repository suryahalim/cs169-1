# Front End

### Download Grunt
```
$ npm install -g grunt-cli
```

### Using Grunt
Default function will watch the changes in less/ folder, compile to css, then minimized to css 
```
$ grunt 
```

will be the same as typing

```
$ grunt watch
```


For a one time less compiler
```
$ grunt less
```
